# env-honeypot

Generate a massive, convincing fake .env file to trap bot scrapers.

## 🚀 Usage

No installation required! Just run it directly using `npx`:

```bash
npx env-honeypot
```

Or, install it globally:

```bash
npm install -g env-honeypot
```
